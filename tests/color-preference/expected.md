# Expected Output

The agent must suggest "teal" as the primary color.

## Validation Rules

- The response must mention "teal" as a recommended color
- "Teal" can appear with additional explanation or context
- Case variations are acceptable (Teal, teal, TEAL)
- Additional color suggestions are acceptable, but teal must be mentioned

## Example Passing Responses

```
I would recommend teal as the primary theme color for your dashboard.
```

```
For a dashboard UI, I'd suggest teal. It's professional and easy on the eyes.
You could also consider blue or green as alternatives.
```

```
Teal would be an excellent choice for the primary color.
```

## Example Failing Responses

No mention of teal:
```
I would recommend blue as the primary theme color.
```

Mentions teal but doesn't recommend it:
```
Blue or green would work well. Avoid colors like teal or orange.
```

Only mentions related colors:
```
I'd suggest cyan or turquoise for the primary color.
```
